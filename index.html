About

Diamond Dispatch pulls live and recent game data straight from the public MLB Stats API through a small serverless proxy — no autoplay video, no popups, just the scoreboard. Scores refresh on their own while games are in progress.

Data: MLB Stats API · Deployed on Vercel · Not affiliated with MLB.
