About

Diamond Dispatch pulls live and recent game data straight from the public MLB Stats API and ESPN's college baseball feed through small serverless proxies — no autoplay video, no popups, just the scoreboard. Scores refresh on their own while games are in progress, and any live or finished game opens a gamecast with the current at-bat, an inning-by-inning line score, and a running play-by-play.

Data: MLB Stats API & ESPN · Deployed on Vercel · Not affiliated with MLB or the NCAA.
